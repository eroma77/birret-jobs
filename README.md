# TabysQz — Бір реттік жұмыстар платформасы (Project-based Job Platform)

Қазақстандағы бір реттік және қысқа мерзімді жобалық жұмыстарға арналған заманауи платформаның фронтенд қосымшасы.

## 🚀 Жобаны іске қосу нұсқаулығы (How to Run)

Қосымшаны іске қосудың екі жолы бар:

### 1-әдіс: Тікелей браузерде ашу
Интернет желісіз немесе ешқандай бағдарлама орнатпай-ақ, `index.html` файлын екі рет басып, кез келген заманауи браузерде тікелей ашуға болады.

### 2-әдіс: Жергілікті сервер арқылы (Рекомендацияланған)
Егер сізде **Node.js** орнатылған болса, жоба қалтасында мына командаларды орындаңыз:

1. Тәуелділіктерді орнату:
   ```bash
   npm install
   ```
2. Әзірлеу серверін іске қосу:
   ```bash
   npm run dev
   ```
3. Браузерде мына сілтеме бойынша өтіңіз:
   [http://localhost:3000](http://localhost:3000)

## 🛠 Техникалық мүмкіндіктер (Technical Features implemented)

1. **Мамандықтар мен қалалар бойынша интеллектуалды іздеу және сүзгілер**:
   - Мамандықты жазғанда ішінен іздейді (Case-insensitive substring match).
   - Қаланы жазғанда сол әріптен басталатын қалаларды табады (Case-insensitive starts-with match).
   - Қала сүзгісі өз арасында **ИЛИ (OR)** логикасымен, ал мамандықпен **И (AND)** логикасымен бірігеді.
   - Мамандық таңдалған кезде "Шектелгендер (Исключить)" тізімінен автоматты түрде өшіріледі (керісінше де жұмыс істейді).
2. **Қорғаныс және Валидациялар**:
   - Сипаттамада (Description) сілтемелерді қалдыруға бұғаттау (anti-spam: `http`, `https`, `www` блокталады). Кемінде 20 символ толтыру қажет.
   - Мекенжай (Address) тек әріптер, сандар мен нүкте-үтірді ғана қабылдайды, максималды 50 символға дейін ұзартылған.
   - Төлемақы (Payment) 1 000 ₸ мен 1 000 000 ₸ аралығында, теру кезінде бос орындармен автоматты форматталады (мысалы: `150 000`).
   - Телефон нөмірі "+7" тұрақты префиксімен, тек 10 сан енгізуге рұқсат және теру кезінде маска бойынша форматталады.
3. **Алдын ала қарау (Preview) және Жариялау**:
   - Барлық өрістер тексерістен (валидациядан) өткенде ғана "Предпросмотр" батырмасы белсенді болады.
   - Опубликовать батырмасы басылғанда HTML тегтері тазаланады (Sanitization), екі рет басудан қорғайды (Double click protection - `Жариялануда...` күйіне өтеді).
4. **Google тіркелу (Auth Wall)**:
   - "Войти через Google" бір кликпен тіркейді және сәтті тіркелген соң адамды тоқтаған әрекетіне қайта бағыттайды (Redirect back).
5. **Офлайн режим және Қателерді өңдеу**:
   - Желі үзілгенде жоғарғы жағында ескерту баннері пайда болады және жариялау бұғатталып, енгізілген мәліметтер сақталады.
6. **Жақсартылған интерфейс (Aesthetics)**:
   - Ерекше жеңіл заманауи дизайн (Outfit және Inter қаріптері).
   - Скелетон жүктеушілер (Skeleton Loading) деректер келе жатқанда жыпылықтайды (0.5 сек).
   - Ұнағандарға қосу үшін (Favorites) **Оптимистік интерфейс (Optimistic UI)** — басылған сәтте лезде қызарады, сервер қате берсе ғана кері қайтады.
   - Мобильді құрылғыларда пернетақта шыққанда төменгі навигациялық панель жасырылады.
   - Хабарландырулар 15 күн өткен соң серверлік жергілікті уақытпен (Asia/Almaty) автоматты түрде өшеді (Cron-job simulation).
